🏀 **Basketball Scoreboard App**

A dynamic and interactive Basketball Scoreboard application built using **HTML**, **Tailwind CSS**, and **JavaScript**. This project simulates a real-time scoreboard experience, allowing users to track and update scores for two teams in an intuitive and visually appealing interface.

The app features a clean and responsive UI designed with Tailwind CSS, ensuring a smooth experience across different devices. Users can easily increase or decrease points with dedicated buttons, making it ideal for practice games, local matches, or learning purposes.

⚡ **Key Features:**

* Real-time score updates for both teams
* Interactive buttons to add points (1, 2, or 3)
* Reset functionality to restart the game instantly
* Responsive and modern UI using Tailwind CSS
* Lightweight and fast performance with vanilla JavaScript

* Building responsive layouts with Tailwind CSS
* Structuring clean and maintainable frontend code
* Improving UI/UX design for better user interaction
